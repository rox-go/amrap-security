# Information Gathering


##  Passive Information Gathering

It involves gathering as much information as possible without actively engaging with the target (OSINT).
Examples:
- IP Addresses and DNS information
- Domain names
- Email addresses and social media profiles
- Web technologies
- Subdomains


## Active Information Gathering

It involves gathering as much information as possible by actively engaging with the target.
Examples:
- Network mapping
- Host discovery
- Port scanning
- Service detection and OS detection
