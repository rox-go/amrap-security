# AMRAP Security

## The Philosophy
**AMRAP** is a term from CrossFit that stands for *As Many Reps As Possible*. In this context, it represents the commitment to constant technical practice. This repository serves as a personal training log for cybersecurity research.

---

## Repository Structure

The documentation is organized as follows:

* **📱 Mobile Security:** Focused on Android (OWASP MASVS L1), Frida dynamic analysis, and APK reversing.
* **🕸️ Web App Security:** Covering Client-Side attacks (XSS, CSRF, CORS), Injection, and CMS research.
* **🌐 Network Security:** From Information Gathering to Active Directory and Post-Exploitation.
* **⚡ Cheatsheets:** A collection of refined "one-liners" for Nmap, Metasploit, Mimikatz, and more.
